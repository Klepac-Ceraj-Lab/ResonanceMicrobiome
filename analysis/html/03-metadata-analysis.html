Unique samples

Except for later quality control, we don't actually want to analyze replicates. And we want to focus on only samples collected in genotek tubes, not ethanol (so ones with an F in the second ID slot).

Using the resolve_sampleID function, I can get just the relevant info for filtering on the first sample.

samples = resolve_sampleID(mgxsamples.sample)
